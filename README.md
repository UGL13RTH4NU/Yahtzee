# Yahtzee
An online html score sheet for playing Yahtzee. 
My mom and wife play Yahtzee all the time (usually over the phone). I got tired of wasting ink and paper on score sheeets.
All this does is allow a player to enter their own rolls and their opponents score at the end and automatically calculates the winner.
Should allow from 2-8 player score counts.
Please notify me of any bugs.

Not affiliated with the actual sellers of the Yahtzee game.

# How it works (or is supposed to work)

My wife and mother 'play' 3 games at one time, which allows them to spread out their rolls in a more favroale manner (allowing better oportunities for not having to take 0's in spots).


v2 is meant for 2-8 players. Basically it lets player 1 keep their own score on the 'app' and at the end of the game Player 1 enters the other Player's final score and it should announce the winner.

v3 is supposed to work just like v2 with the added functionality that if you tap a players name at the top, it changes to their score sheet and you can enter scores for them, it auto-calculates everyones scores, and on the main page (Player 1), it gives the winner when all fields are entered.
